Polaris Tiles (0.0.1)

Getting Started

  • Install the repo via NPM npm install --save-dev git+ssh://git@github.com/skyglobal/tiles.git#v0.0.1
  • Import the Tile SCSS into your SCSS using import '../node_modules/tiles/src/tile';
  • Get the markup for the required tile by clicking it and copying the generated markup from the console.
  • The polaris tiles have 3 breakpoints; Tablet <1024, Mobile <540. Font sizes change at each breakpoint.
  • Use the grid system of your choice to control the width of the tiles. A basic grid system is used for demo purposes only.
  • Note: This is only intended as a first step to share styles to meet Technology deadlines. A more comprehensive toolkit will replace this repo.
